# TextTranscribe

TextTranscribe is a web application designed to convert PDF documents into editable text using advanced Optical Character Recognition (OCR) techniques. The application leverages Tesseract OCR and Poppler to provide accurate and efficient text extraction, supporting multiple languages and various document formats.

## Features

- **PDF to Text Conversion:** Extract text from PDF documents and images.
- **Multi-Language Support:** Recognize and extract text in multiple languages.
- **Error Handling:** Robust preprocessing and error correction for improved accuracy.
- **User-Friendly Interface:** Simple and intuitive web interface for file upload and text extraction.

## Technologies Used

- **Flask:** Web framework for the backend.
- **Bootstrap:** For responsive and modern UI design.
- **Tesseract OCR:** Open-source OCR engine for text recognition.
- **Poppler:** Library for converting PDFs into images.
- **PDF2Image:** Tool to convert PDF pages into images for OCR processing.
